# js-plugin-cli

> 开发 js 插件的 webpack 配置，方便开发什么 js 插件的时候，进行开发、打包等操作。

## 使用

```
// clone project
git clone git@github.com:rni-l/plugins.git

cd cli/js-plugin

// install dependencies
npm i // or cnpm i

// edit plugin name
vim package.json // edit pluginName

// develop
npm run dev

// build
npm run build
```

开发的文件在 src 里面

打包之后，出现在 dist 里面
